Welcome!

This LabintheWild study is a demonstration. It is designed to be a template for building other studies using the jsPsych library and LabintheWild utilities.

Although this template only demonstrates a single kind of stimulus, other trial types are possible. See the jsPsych documentation to see which other trial types are supported.

Also, visit our tutorial to learn about best practices for designing large-scale, uncompensated studies.
